# IPL-Dashboard

IPL Analysis Dashboard (2008–2025)

📄 Description
This project is an interactive IPL (Indian Premier League) Analysis Dashboard built using Power BI to visualize team and player performance across seasons from 2008 to 2025.
It provides detailed insights into match statistics, player achievements, and team standings through dynamic filters and visually rich reports.

🎯 Purpose
The main purpose of this project is to:

Analyze IPL data in an interactive way
Identify top-performing players and teams
Track season-wise trends and performance
Improve data visualization and analytical skills using Power BI

🛠️ Tech Stack
Power BI (Data Visualization)
DAX (Data Analysis Expressions)
Power Query (ETL)
Data Modeling
📥 Data Source
IPL dataset (ball-by-ball and match-level data in CSV format)
Public datasets and YouTube tutorial references

⭐ Features/📊 Key Highlights

📅 Season Filter – Select any IPL season dynamically
🏆 Champions & Runner-Up – Final match results
🔢 Overall Stats
Total Matches
Total 4’s & 6’s
Centuries & Half-Centuries
Total Teams & Venues
🟠 Orange Cap Analysis
Top run scorer
Total runs
Team details
🟣 Purple Cap Analysis
Highest wicket-taker
Total wickets
Team details
💥 Batting Insights
Most Fours
Most Sixes
📋 Points Table
Matches Played
Wins / Losses
Net Results
Total Points
